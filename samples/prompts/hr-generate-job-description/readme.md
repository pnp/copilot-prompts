# 🚀 Generate Job Description Using Copilot 📅

[Demo video](./assets/HR_Prompt_Demo.mp4)

## Summary

This prompt helps HR teams and hiring managers generate a clear, role-specific job description by combining role title, required skills, responsibilities, qualifications, and regional hiring expectations.

## Prompt 💡

Prepare a job description for a [Position Name] with expertise in [Skills], [outlining key responsibilities], required [qualifications], and senior-level expectations for the [Country Name] hiring market.

## Description ℹ️

Use this prompt to quickly draft a professional, structured job description tailored to the role and market context.

## Instructions 📝

1. Make sure you have Copilot for Microsoft 365 in your tenant
2. Open the Copilot app in Teams or Microsoft 365
3. Copy and paste the prompt
4. Replace placeholders like [Position Name], [Skills], [qualifications], and [Country Name] with your hiring details

### Improvise Usage 🚀

Ask Copilot to generate multiple variants for junior, mid-level, and senior roles, then compare and refine the best output.
