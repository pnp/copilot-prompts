# "Action Plan" Minutes After Operational Meetings


## Summary

This prompt generates "action plan" style minutes from operational meetings, helping you leave meetings with a clear, executable plan.

## Prompt 💡

Generate "action plan" style minutes using the following structure:

- Very brief summary of the topics discussed.

- List of decisions made.

- List of actions including:
  - Description
  - Owner
  - Target date
  - Dependencies (if any)
  - Identified risks or blockers

- Next follow-up date.

Use a very practical, execution-focused tone.

## Description ℹ️

This prompt helps you capture structured action plan minutes from operational meetings in Microsoft Teams. It produces a clear summary covering topics discussed, decisions made, detailed action items with owners and timelines, and the next follow-up date using a practical, execution-focused tone.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the operational meeting.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated action plan and distribute to team members.

### Improvise Usage 🚀

You can customize this prompt to include priority levels for actions, budget impact notes, or integration with project management tools like Planner or Azure DevOps.
