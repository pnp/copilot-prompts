# Risk Analysis and Assessment Meeting Minutes



## Summary

This prompt generates risk analysis and assessment minutes, helping manage risks in an organized way especially in IT or project environments.

## Prompt 💡

Generate risk analysis and assessment minutes using the following structure:

- List of identified risks.

For each risk:
  - Description
  - Impact (low, medium, high)
  - Probability (low, medium, high)
  - Proposed mitigation
  - Responsible for follow-up

- Summary of the most critical risks.
- Next actions for reviewing or updating risks.

Use a clear, risk-management-oriented tone.

## Description ℹ️

This prompt helps you capture structured minutes from risk analysis and assessment meetings in Microsoft Teams. It produces a clear summary covering identified risks with their impact, probability, proposed mitigations, responsible parties, and next actions using a risk-management-oriented tone.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the risk analysis meeting.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated minutes and update the risk register accordingly.

### Improvise Usage 🚀

You can extend this prompt to include risk categories (technical, financial, operational), risk scores, or heat map data depending on your risk management framework.
