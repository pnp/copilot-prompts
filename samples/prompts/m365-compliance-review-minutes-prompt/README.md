# Internal Control and Compliance Review Minutes



## Summary

This prompt generates internal control and compliance review minutes, recording findings, evidence, and actions related to compliance or internal audit meetings.

## Prompt 💡

Generate internal control and compliance review minutes using the following structure:

- Scope of the review (processes, areas, systems).
- Findings identified, with a brief description.
- Evidence mentioned during the meeting.
- Classification of each finding (critical, high, medium, low).
- Proposed corrective actions, including owners and estimated timelines.
- Associated risks if actions are not implemented.

Use a formal and precise tone.

## Description ℹ️

This prompt helps you capture structured minutes from internal control and compliance review meetings in Microsoft Teams. It produces a clear summary covering the review scope, findings, evidence, classifications, corrective actions, and associated risks using a formal and precise tone.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the compliance review meeting.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated minutes and distribute to relevant stakeholders.

### Improvise Usage 🚀

You can adapt this prompt for different compliance frameworks (SOX, ISO, GDPR) or add sections for regulatory references and audit trail documentation.
