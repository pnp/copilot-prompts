# Team Retrospective or Evaluation Minutes


## Summary

This prompt generates minutes of a team retrospective or evaluation, capturing lessons learned after a sprint, project, or major phase.

## Prompt 💡

Generate the minutes of a team retrospective or evaluation using the following structure:

- What worked well.
- What did not work or blocked progress.
- Improvement ideas proposed by the team.
- Concrete actions to be implemented.

## Description ℹ️

This prompt helps you capture structured minutes from team retrospective or evaluation meetings in Microsoft Teams. It produces a clear summary covering what went well, what didn't work, improvement ideas, and concrete actions to implement going forward.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the retrospective meeting.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated minutes and share with the team.

### Improvise Usage 🚀

You can extend this prompt to include team satisfaction scores, comparisons with previous retrospectives, or categorization of improvement actions by priority.
