# Meeting Minutes with External Stakeholders



## Summary

This prompt generates minutes oriented to external stakeholders, documenting conversations with leadership, clients, or other departments while clearly capturing expectations and commitments.

## Prompt 💡

Generate minutes oriented to external stakeholders using the following structure:

- Purpose of the meeting and participants.
- Topics discussed from the stakeholder's perspective.
- Stakeholder needs, expectations, or objectives.
- Commitments made on our side.
- Commitments made by the stakeholder (if any).
- Next relevant milestone (meeting, delivery, decision).

Use a clear, diplomatic tone suitable for sharing by email.

## Description ℹ️

This prompt helps you capture structured minutes from meetings with external stakeholders in Microsoft Teams. It produces a clear summary covering the meeting purpose, topics discussed, stakeholder needs, commitments from both sides, and next milestones using a diplomatic tone suitable for email sharing.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the meeting where you need the minutes.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated minutes and share with stakeholders via email.

### Improvise Usage 🚀

You can adapt this prompt for different types of stakeholders such as vendors, partners, or board members by adding sections relevant to each relationship type.
