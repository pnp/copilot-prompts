# Training Session or Technical Workshop Minutes



## Summary

This prompt generates a summary of a training session or technical workshop, capturing key knowledge including topics covered, examples shown, Q&A, and next steps.

## Prompt 💡

Generate a summary of a training session or technical workshop with the following structure:

- Session objective and target audience.
- Main topics covered, with a brief description of each.
- Practical examples or demos shown.
- Frequently asked questions and the answers provided.
- Recommendations or additional resources mentioned.
- Suggested next steps after the session.

Use an educational and learning-oriented tone.

## Description ℹ️

This prompt helps you capture structured summaries from training sessions or technical workshops in Microsoft Teams. It produces a clear summary covering the session objective, topics covered, demos, Q&A, resources, and next steps using an educational and learning-oriented tone.

## Instructions 📝

1. Ensure you have Copilot for Microsoft 365 enabled in your tenant.
2. Open the Microsoft Teams app.
3. Join or open the training session or workshop meeting.
4. Access the Copilot app within Teams.
5. Paste the prompt in the Copilot app.
6. Review the generated summary and share with attendees.

### Improvise Usage 🚀

You can customize this prompt to focus on specific areas like hands-on lab outcomes, certification preparation notes, or skill assessments depending on the type of training.
