# 🚀 Last 7 Days Recap for Manager Updates 📅

[Demo video](./assets/Demo.mp4)

## Summary

This prompt helps create a concise weekly recap from the last 7 days of emails, meetings, and Teams chats so you can quickly share status with your manager.

## Prompt 💡

Scan my last 7 days of emails, meetings, and Teams chats. Draft a concise status update for my manager covering: key wins, decisions made, blockers needing his input, and what I'm focused on next week.

## Description ℹ️

Use this prompt to generate a structured weekly status update with key wins, decisions made, blockers, and next-week focus areas.

## Instructions 📝

1. Make sure you have Copilot for Microsoft 365 in your tenant
2. Open the Copilot app in Teams or Microsoft 365
3. Copy and paste the prompt
4. Ask Copilot to refine the output tone and level of detail for your manager

### Improvise Usage 🚀

Ask Copilot to produce two variants: a short executive summary and a detailed version with action items.
