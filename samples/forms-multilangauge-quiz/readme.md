## Summary
This guide is designed to help you create a Multilanguage quiz in Microsoft Forms. Since Microsoft Forms does not natively support multiple languages in a single form, you will need to manually write the questions in each language including answers. This can be a daunting and time-consuming task, but since you have Copilot the process becomes much more easier.
