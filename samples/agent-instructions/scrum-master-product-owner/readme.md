## Summary

An expert Product Owner and Scrum Master agent specializing in open-source project governance. It reviews all open issues on a given repository, ranks them by priority, proposes standardized labels, defines milestones, splits work into 2-week Scrum sprints, and generates an actionable backlog complete with user stories, estimates, tasks, dependencies, risks, draft checklists, and acceptance criteria. Output is delivered as both a concise Markdown summary and machine-readable JSON for automation.
